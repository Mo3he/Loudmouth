# Loudmouth

A powerful, source-agnostic music player for iPhone and iPad.

## Features

- Play music from local files, iCloud Drive, Subsonic/Navidrome servers, NAS/DLNA, Backblaze B2, Dropbox, Google Drive, and OneDrive
- Gapless playback with crossfade
- 10-band parametric EQ with custom presets
- Synced lyrics via LRC files or LRCLIB
- ReplayGain / EBU R128 volume normalisation
- Last.fm and ListenBrainz scrobbling
- Smart playlists
- Offline cache / pin sources for offline listening
- Wi-Fi file transfer
- AirPlay and Bluetooth output routing
- Tag editor (ID3, Vorbis, M4A)
- Lock screen / Control Centre integration
- Home Screen widget
